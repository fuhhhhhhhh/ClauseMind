# ClauseMind 截图目录说明

将课程设计最终使用的系统截图统一放在本目录下，建议命名如下：

- `01-login.png`
- `02-register.png`
- `03-dashboard.png`
- `04-upload.png`
- `05-contract-list.png`
- `06-contract-detail.png`
- `07-parse-result.png`
- `08-normalized-result.png`
- `09-review-progress.png`
- `10-risk-analysis.png`
- `11-suggestion.png`
- `12-report.png`
- `13-admin.png`

建议要求：

1. 分辨率尽量统一，优先横向截图。
2. 每张图尽量只截一个核心功能区域，不要带太多桌面背景。
3. 登录、上传、解析、审查、报告、管理员后台这几张必须保留。
4. 如果页面内容过长，优先截首屏关键信息，必要时分两张图。

后续如果需要自动生成 Word，可直接按以上文件名读取。
